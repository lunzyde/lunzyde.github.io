C:/Documents/nil/sole/fey/run.exe

Completely developed by myself!
Puerto Rico Online Game Jam 2019
  • Roles: 
  • Visual Scripting Game Design Sound Design Pixel Art
  • Tools: 
  • Construct 2 FL Studio Photoshop Elements
  • Description: Short and spooky puzzle-maze experience. There are keys to get and doors to unlock, but your flashlight that the battery drains overtime.
  • Developed the game with its simple mechanics in less than 24 hours. After the game jam, I worked on a few updates to improve the gameplay experience.
  • Designed echoey menacing sounds and ambiances to fright players.
  • Made pixel art assets to take advantage of multiple visual filters, taking inspiration from retro psychological horror games and old malfunctioning CRT monitors.
  • Itch.io
  • Source Code

GastroMaster's Tour:
A Humble Beginning

by Hangry Squad Games
Latinx In Gaming's Unidos Online Game Jam 2020
  • Roles: 
  • Sound Design Music Production Pixel Art
  • Tools: 
  • FL Studio Photoshop Elements
  • Description: Food truck cooking game in which you make mofongos. As the game progresses, customers will ask you for different types of mofongo to cook up.
  • Recorded food, drinks, and kitchen tools to create multiple cooking sound effects.
  • Composed and produced five Puerto Rican and Latin American inspired loops.
  • Additionally, I helped with some pixel art assets and sketched the team logo.
  • Itch.io
  • Soundtrack

RRR: Rival Relationship Repairs

Completely developed by myself!
Global Game Jam 2020 - Puerto Rico
  • Roles: 
  • Visual Scripting Game Design Sound Design Pixel Art
  • Tools: 
  • Construct 3 FL Studio Photoshop Elements
  • Description: Local co-op game inspired by Asteroids. It contains a short story, and there is only one button for each player to use their character's spaceship.
  • Focused on the gameplay experience and performed multiple playtest sessions.
  • Designed the one-shot sounds with synths, including the loops for the idle and movement state of each player (a primitive experiment with adaptive music).
  • Created pixel art assets to fit the saturated and illuminated retro arcade aesthetic.
  • Itch.io
  • Source Code

FROST-BOUND

Global Game Jam 2021 - Puerto Rico
  • Roles: 
  • Blueprints Level Design Niagara
  • Tools: 
  • Unreal Engine 4
  • Description: Short, vehicular adventure game, set in the icy terrain of Antarctica. You get tasked with driving a scientist in critical condition to a rescue helicopter, all while facing the harsh weather conditions.
  • Made a simple snow particle emitter to attach on blueprint actors, but also experimented with the gloomy lighting and foggy environments.
  • Created the levels with the landscape tool and marketplace assets.
  • Additionally, I assisted the music composer with blueprints to change the progression of the music with invisible volumes that trigger when overlapped.
  • Itch.io
  • Source Code
